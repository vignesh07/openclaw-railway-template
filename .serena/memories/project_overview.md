OpenClaw Railway template: a Dockerized Node/Express wrapper around OpenClaw for Railway deployment. It serves a setup UI at /setup, protects /setup and the Control UI with HTTP Basic auth via SETUP_PASSWORD, persists state/workspace under /data, starts/proxies the OpenClaw gateway, and exposes health endpoints. Current source layout is small: src/server.js is the wrapper server and src/setup-app.js is the browser setup wizard script.
