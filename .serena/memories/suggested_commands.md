Core commands: `npm run dev` or `npm start` to run wrapper server, `npm run lint` for syntax check of src/server.js, `npm test` for Node test runner, `npm run smoke` for smoke test script. Local container smoke path in README: `docker build -t clawdbot-railway-template .` then `docker run --rm -p 8080:8080 -e PORT=8080 -e SETUP_PASSWORD=test -e OPENCLAW_STATE_DIR=/data/.openclaw -e OPENCLAW_WORKSPACE_DIR=/data/workspace -v $(pwd)/.tmpdata:/data clawdbot-railway-template`.
